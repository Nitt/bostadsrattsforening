# Användarmanual - Bostadsrättsföreningens Webbplats

## Välkommen! 👋

Denna webbplats är designad för att göra det enkelt för alla medlemmar i bostadsrättsföreningen att hantera och komma åt föreningsinformation. Du behöver inga tekniska kunskaper för att använda den!

## Kom igång

### Öppna webbplatsen
1. Öppna din webbläsare (Chrome, Firefox, Safari, Edge)
2. Gå till webbplatsens adress
3. Du kommer att se startsidan med tre huvudfunktioner

### Navigering
- **Meny överst**: Klicka på länkarna för att växla mellan sidorna
- **På mobil**: Klicka på hamburgermenyn (☰) för att se navigationslänkarna

## Huvudfunktioner

### 🏠 Uppdatera Poster
**Vad det är:** Här kan du lägga till och redigera information om lägenheter, möten, underhåll och ekonomi.

#### Så här använder du det:
1. Klicka på "Uppdatera Poster" i menyn
2. Välj typ av post du vill skapa:
   - **Lägenhet**: För information om lägenheter och ägare
   - **Stämma/Möte**: För att planera möten
   - **Underhåll**: För underhållsärenden
   - **Ekonomi**: För ekonomiska poster

3. Fyll i formuläret:
   - Alla fält markerade med * är obligatoriska
   - Fyll i så mycket information som möjligt
   - Du kan alltid redigera senare

4. Klicka på "Spara" knappen
5. Du får en bekräftelse när det är sparat
6. Din post visas i listan "Senaste poster"

#### Tips:
- ✅ Kontrollera att all information är korrekt innan du sparar
- ✅ Använd tydliga beskrivningar
- ✅ Du kan redigera poster genom att klicka på ✏️ ikonen

### 🏢 3D Kvarter
**Vad det är:** En visuell representation av vårt bostadsområde.

#### Så här använder du det:
1. Klicka på "3D Kvarter" i menyn
2. Du ser antingen:
   - En interaktiv 3D-vy (om alla bibliotek laddas)
   - En förenklad 2D-vy med ikoner och byggnader

3. **Kontroller:**
   - **Översikt**: Visa hela området
   - **Byggnader**: Fokusera på byggnaderna
   - **Lägenheter**: Highlight lägenheter
   - **Parkering**: Visa parkeringsområden
   - **Trädgård**: Fokusera på grönområden

4. **Interaktion:**
   - Klicka på byggnader för att se information
   - Använd kontrollknapparna för olika vyer

### 📄 Dokument
**Vad det är:** Här hittar du och kan ladda upp viktiga dokument som stadgar, protokoll och ekonomiska rapporter.

#### Ladda upp dokument:
1. Klicka på "Dokument" i menyn
2. Scrolla ner till "Ladda upp nytt dokument"
3. Fyll i:
   - **Kategori**: Välj rätt typ av dokument
   - **Titel**: Ge dokumentet ett tydligt namn
   - **Beskrivning**: Beskriv kort vad dokumentet innehåller
4. Lägg till filen:
   - Klicka på upload-området ELLER
   - Dra och släpp filen direkt
5. Klicka "Ladda upp dokument"

#### Hitta dokument:
1. **Kategorier**: Klicka på kategori-flikarna för att filtrera
2. **Sök**: Använd sökrutan för att hitta specifika dokument
3. **Sortera**: Ändra sorteringsordning med rullgardinsmenyn

#### Hantera dokument:
- **📥 Ladda ner**: Klicka för att ladda ner dokumentet
- **👁️ Förhandsgranska**: Se dokumentet utan att ladda ner
- **✏️ Redigera**: Ändra dokumentinformation

## Vanliga frågor (FAQ)

### Jag har gjort ett misstag, kan jag ändra det?
✅ **Ja!** Du kan redigera de flesta poster och dokument genom att klicka på redigeringsikonen (✏️).

### Vad händer om jag råkar ta bort något?
⚠️ **Försiktighet:** När du tar bort något visas en bekräftelse. Läs den noggrant innan du bekräftar.

### Kan jag använda webbplatsen på min telefon?
✅ **Absolut!** Webbplatsen fungerar på alla enheter - datorer, surfplattor och smartphones.

### Vilka filformat kan jag ladda upp?
📎 **Stödda format:**
- PDF (.pdf)
- Word-dokument (.doc, .docx)
- Excel-filer (.xls, .xlsx)
- Max filstorlek: 10MB

### Jag ser felmeddelanden, vad ska jag göra?
1. **Kontrollera internetanslutningen**
2. **Ladda om sidan** (F5 eller Ctrl+R)
3. **Kontrollera att alla obligatoriska fält är ifyllda**
4. **Kontakta styrelsen** om problemet kvarstår
